# Rubik's Cube

Yet another implementation, this time using [Golang bindings](https://github.com/gen2brain/raylib-go)
for [RayLib](https://github.com/raysan5/raylib). 

# How To Deploy

1. On TrimUI find System Tools > Network > Display IP
2. Type `export IP=<IP>`
3. Optional: start Dummy app on TrimUI
4. Run `make all`
