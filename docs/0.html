Controls
Desktop
  • 'H' — help on/off
  • ←, ↑ or → keys to rotate
  • 1-9 to (de)select faces
  • hold 'S' to shuffle
  • Left Control' (hold) + Up/Down to rotate around the Z-axis
TrimUI Smart Pro
  • Y to enter/exit select mode
  • use joysticks to select/rotate
  • A/B to rotate
  • hold X to shuffle
  • up/down+start for Z-axis
  • menu+start -> exit
